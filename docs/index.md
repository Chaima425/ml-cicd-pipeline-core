# Documentation du projet "ML CICD Pipeline Core"

Ce projet présente la mise en place d’un pipeline CI/CD complet autour d’un modèle de Machine Learning, avec :

- **Backend** : FastAPI
- **Frontend** : Streamlit
- **Tracking ML** : MLflow
- **Tests & Qualité** : Pytest
- **Dockerisation**
- **Pipeline GitHub Actions**
- **Déploiement (Azure / Docker Hub)**

L’objectif : comprendre **à quoi sert un pipeline CI/CD** et comment l’appliquer dans un projet ML/IA moderne.
